# List

> 데이터에 순서가 매겨지고(ordered)하고 중복된 데이터를 허용하는 자료구조.

## Array list

| 구분 | 시간복잡도 |
| --- | -------- |
| 조회 |          |
| 순회 |          |

### 장점

- 데이터의 참조가 쉽다.

### 단점

- 데이터 삭제 과정에서 데이터의 복사가 빈번히 발생한다.[^arraylist_disadvantage]

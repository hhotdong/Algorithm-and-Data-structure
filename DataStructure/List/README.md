# List

> 데이터에 순서가 매겨지고(ordered)하고 중복된 데이터를 허용하는 자료구조.

## 주요 동작

- Append: 리스트 맨 끝에 원소 추가
- Insert: 리스트 특정 위치에 원소 추가
- Remove: 리스트에서 특정 데이터 삭제
- Search: 리스트에서 특정 데이터 탐색
- Traverse: 리스트의 데이터를 하나씩 순회
